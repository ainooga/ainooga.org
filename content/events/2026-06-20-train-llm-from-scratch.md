## About the workshop

Language models power everything from chatbots to code assistants. But how do they *actually* work? In this full-day workshop, we'll demystify the stack by building one from scratch.

### What you'll learn

- **Tokenization** — how text becomes numbers
- **Architecture** — transformer basics, attention mechanisms
- **Training loop** — forward pass, backpropagation, loss curves
- **Evaluation** — perplexity, generation quality, failure modes
- **Inference** — serving a model, prompt engineering basics

### Prerequisites

- Python 3.10+ installed
- Basic familiarity with Python (loops, functions, arrays)
- A laptop (power and WiFi provided)

No prior machine learning experience required. We'll provide a Jupyter notebook environment.
